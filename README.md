# cadence-visualizer

Visualizes node movement over time from a SQLite geolocation dataset.

## Setup

**1. Export frames from your database**

Copy your SQLite database into `db/` (generated by your simulator software), then run:

```bash
python export_frames.py --db db/your_database.db --out frames/frames.jsonl.gz
```

Both `db/` and `frames/` are gitignored — you'll need to create them and populate them manually.

**2. Run the visualizer**

```bash
cd app
npm install
npm run dev
```

Open `http://localhost:5173`, then load your `frames/frames.jsonl.gz` file.

## Features

- Canvas renderer with colored nodes and motion trails
- Timeline scrubber with play/pause and speed control
- Node filter panel — search, highlight (others grayed out), or isolate specific nodes
